# Java Learning – Conditional Statements Programs

This repository contains basic Java programs created while learning **Java programming fundamentals**, mainly focused on **Conditional Statements** and decision-making concepts.

## Topics Covered

The programs in this repository cover:

- If Statement
- If-Else Statement
- Else-If Ladder
- Switch Statement
- Ternary Operator
- Conditional Logic Practice Problems

## Programs Included

1. Calculator  
   - Performs basic arithmetic operations like addition, subtraction, multiplication, and division.

2. Else If  
   - Demonstrates the use of else-if ladder in Java.

3. Even Odd  
   - Checks whether a number is even or odd.

4. If Else  
   - Basic example of if-else condition.

5. Income Tax Calculator  
   - Calculates tax based on income range using conditions.

6. Largest of 2  
   - Finds the larger number between two numbers.

7. Largest of 3  
   - Finds the largest among three numbers.

8. Pass or Fail  
   - Checks result status based on marks.

9. Problem 1  
   - Practice problem using conditional logic.

10. Problem 2  
   - Practice problem using conditional logic.

11. Problem 3  
   - Practice problem using conditional logic.

12. Problem 4  
   - Practice problem using conditional logic.

13. Switch Statement  
   - Uses switch-case for multiple conditions.

14. Ternary Operator  
   - Demonstrates shorthand conditional operator.
